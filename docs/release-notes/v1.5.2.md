## What's new

- **Smoother terminal resizing** — The terminal prompt now stays visible while shrinking the terminal panel.
- **Terminal focus shortcut** — **Ctrl+'** now switches focus between the editor and terminal.
- **Tidier terminal panels** — Closing the last terminal tab now hides the terminal side when it is the only panel there.
- **Cleaner recent items** — Recent folders and workspaces display more consistently, and missing recent entries are handled more gracefully.
- **Tab and menu polish** — Editor tabs now have a close context menu, and the File menu has been simplified.
