## What's new

- **Faster large files** — Opening, editing, wrapping, and searching large documents now feels much more responsive.
- **Broader syntax highlighting** — Volt now includes built-in highlighting for many more languages and file types, including C, C++, C#, CSS, HTML, JavaScript, JSON, Python, Rust, SQL, TypeScript, XML, YAML, and more.
- **Smarter HTML highlighting** — CSS inside `<style>` blocks and JavaScript inside `<script>` blocks are highlighted with the right language rules.
- **Clearer theme names** — Bundled themes now use their familiar upstream names, such as Atom One, GitHub, Dracula, Nord, PowerShell ISE, and C/C++.
